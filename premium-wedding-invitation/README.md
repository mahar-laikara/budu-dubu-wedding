# Premium Digital Wedding Invitation & QR RSVP System

A premium, luxury-styled digital wedding invitation designed for the wedding of **Budu & Dubu**.

## Features

- 🌸 **Luxury Visuals**: Sleek dark emerald green and gold foil design with serif typography (*Cinzel* and *Playfair Display*).
- 🎵 **Ambient Background Music**: A high-speed, direct Chopin piano ballad loops gently, controllable via a responsive volume button on the cover.
- ✨ **Scratch-to-Reveal**: Interactive canvas scratch card allowing guests to scratch and reveal the wedding date and location details.
- ⏳ **Countdown Clock**: Real-time ticker counting down the days, hours, minutes, and seconds to September 27, 2026.
- 📅 **Interactive Timeline**: Staggered, scroll-animated program schedule showing the ceremony and dinner program.
- 🎟️ **QR Code Entrance Ticket**: Generates a dynamic guest entry pass QR code upon submitting the RSVP form.
- 🔍 **Organizer Check-in Dashboard**: Reintegrated dashboard right at the bottom of the invitation page (secured by passcode `1234`) with:
  - **Live Camera Scanner**: Scans guest QR passes at the entrance (with custom beeps).
  - **Registry Database**: Lists all RSVP status and check-ins.
  - **Ticket Drawer**: View and print passes for any registered guest.

## Technologies Used

- **HTML5** & **Vanilla CSS**
- **JavaScript**
- **qrcode.js** (QR generation)
- **html5-qrcode** (Camera QR decoding)

## How to Deploy on GitHub Pages

1. Create a new public repository on GitHub.
2. Click **"uploading an existing file"** and drag all files from this directory (including `assets/`) into your repository.
3. Go to **Settings** -> **Pages** -> under **Branch**, select `main` (or `master`) and click **Save**.
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`!
